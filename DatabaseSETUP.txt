Database Setup Instructions!

1. Left click on the Wamp icon in your tray.

2. Open "phpMyAdmin".

3. Sign in with: Username = root
				 Password = 
				 *No password by default. Leave blank.
				 
4. On the left-size pannel, create a new Database.

5. I named mine "4430_project" with "latin1_swedish_ci".

6. Once database is created and selected, select "IMPORT" at the top.

7. Click "Choose File" and select the "DatabaseRoughDraft.sql" file included in this commit.

8. Hit "GO" at the bottom.

9. You should now have a few tables with 1 row and test data in each.
